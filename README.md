# 🧠 Smart Design

**Smart Design** is a **macOS application** built with **Swift** that helps calculate and recommend optimal **reading room features** based on the **number of people** in the room.  

By entering the number of occupants, the app provides smart suggestions for parameters such as:  
- 💡 **Lighting**  
- 🖥️ **Screen size**  
- 🪑 **Seating arrangement**  
- 🔊 **Sound layout**  

These recommendations aim to create a **comfortable**, **efficient**, and **intelligently designed** reading environment.
